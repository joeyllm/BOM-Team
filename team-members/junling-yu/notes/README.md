# Notes 📝

This directory is for reading notes, architecture ideas, and daily logs.

## Files

- [DataNotebooksinPyEarthTool.md](DataNotebooksinPyEarthTool.md) — records PyEarthTools data indexing, cataloging, custom indexers, time series retrieval, pattern indexing, transforms, and operations.

- [Graphcast.md](Graphcast.md) — describes deterministic graph-based weather forecasting, with node/edge representation, message passing, encoder-processor-decoder design, and autoregressive rollout.
- [RadarVisualisation.md](RadarVisualisation.md) — records radar visualization techniques, including 2D reflectivity mapping and 3D storm structure rendering.
