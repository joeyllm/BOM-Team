# 📊 Semester 1 – Data and Infrastructure

The first semester focuses on **understanding the infrastructure and preparing forecasting datasets**.

Key activities include:

### Data Preparation

Students will work with project data already stored on the remote system.

Tasks include:

- exploring the dataset
- validating source quality
- cleaning and normalising records
- preparing data suitable for machine learning

The goal is to produce **high-quality validated datasets** that can be used for forecast modelling.

---

### Baseline Analysis

Near the end of the first semester the team will begin building **baseline analytical and predictive models**.

These workflows may analyse data to determine characteristics such as:

- spatial coverage
- temporal patterns
- other useful metadata

These tools help organize large datasets and enable more targeted modelling later.
