# 📋 Project Overview

The project is divided into **two main semesters**, each focusing on different stages of the workflow used to build and evaluate nowcasting systems.

Students will work with:

- weather and observational datasets
- data engineering and machine learning workflows
- GPU compute infrastructure
- forecast modelling and evaluation

The goal is to learn how modern nowcasting systems are built **from raw data through to validated forecasts**.
